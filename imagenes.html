Imágenes en HTML

LAs etiquetas utilizadas para el manejo de imágenes en HTML son las siguientes:

Etiqueta <"img">
Indica que se van a manejar imágenes.


Atributos
SRC = Nombre del archivo

height = altura

width = ancho

alt = texto alternativo

Ejemplo <"img src="fondo.jpg" alt="wallpaper" height="400" width="600px"
