HTML(Hyper Text Markup Language)

Las páginas HTML son documentos de texto. HTML usa elementos para definir la estructura de la página, estos elementos se definen por etiquetas encerradas en parentesis angulares < >.

El contenido de un elemento se encierra entre etiquetas.Existen etiquetas de apertura(<"etiqueta">) y de cierre (<"/etiqueta">). El nombre de la etiqueta debe ser el mismo y se distingue a las de cierre con un "/".

Las etiquetas pueden tener atributos los cuales se especifican en la etiqueta de inicio.
