Texto en HTML

Para manejar texto mediante HTML utilizamos las siguintes etiquetas:

Ecabezados
Se utilizan las etiquetas <"h1"> / <"h6">,teniendo mayor importancia <"h1"> y menos importancia<"h6">

Párrafos
Se utiliza la etiqueta <"p"> para crear un nuevo Párrafo.

Saltos de línea
Se utiliza la etiqueta <"br"> para separar un parrafo de otro.

Espacios en blanco
Se utiliza " " para indicar que hay espacios en blanco en el texto.

Negritas
Se utiliza la etiqueta <"b"> para remarcar en negritas un texto.

Cursivas
Se utiliza la etiqueta <"i"> para darle un toque de cursiva al texto.
