Tablas en HTML

Para manejar tablas mediante HTML utilizamos las siguientes etiquetas:

Tabla <table
Indica el principio y fin de la tabla

Descripción <caption>
Se usa para describir la función de la tabla

Encabezado <thead>
Para el título de la tabla

Pie de página <foot>
Para eagregar un pie de página

Cuerpo de página <tbody>
Para agregar contenido a la tabla

Fila <tr> (table row)
Para agregar una fila en la tabla

Celda del encabezado <th>
Representra la primer celda de una fila

Celda del contenido <td>
Para las celdas de la tabla
