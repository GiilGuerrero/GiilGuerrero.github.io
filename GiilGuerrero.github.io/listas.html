Listas en HTML

Cada lista en HTML está compuesta por elementos y pueden ser de 3 tipos:

Ordenadas
Utiliza la etiqueta <"ol"> para crearla

Utiliza la etiqueta <"li"> para especificar los elementos.

No Ordenadas

Utiliza la etiqueta <"ul"> para crearla

Utiliza la etiqueta <"li"> para especificar los elementos.

De descripción
Utiliza la etiqueta <"dl"> para crearla

Utiliza las etiquetas <"dt"> y <"dd"> para especificar los elementos.

Listas ordenadas

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Listas no ordenadas

  • Primer elemento
  • Segundo elemento
  • Tercer elemento

Listas de descripción
